# Online Learning Platform

This project is a simple and attractive static website for an online learning platform. It is designed to help users explore educational courses, learn about the platform, enroll in classes, and access a login experience in a smooth and user-friendly way.

## Project Overview
This website consists of multiple HTML pages that work together to provide a complete online learning experience:

- Home page: introduces the platform and encourages visitors to get started.
- About page: explains the purpose, mission, and value of the platform.
- Courses page: showcases available learning programs and opportunities.
- Enrollment page: allows users to register for courses.
- Login page: provides a simple sign-in interface for users.
- Thank you page: displays a confirmation message after successful interaction.

## Key Features
- Clean and modern design
- Easy navigation between pages
- Structured course-focused layout
- Simple styling with CSS
- User-friendly interface for learning and enrollment

## How to Run
1. Open the project folder in your browser.
2. Open any HTML file, such as project1.html, to view the website.
3. You can also use a simple local web server if needed.
